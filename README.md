stepbar
=======

Clean step bar that shows you where you are in the order progress

![Preview stepbar](http://andersonba.com/stepbar/stepbar.jpg)

    <div class="stepbar" data-steps='["Start", "Step 2", "Step 3", "Step 4", "Completed"]'></div>

    // change step
    stepbar.step(4);

    // next step
    stepbar.next();

    // previous step
    stepbar.prev();
